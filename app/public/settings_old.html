Chaves de API dos Modelos de IA
🔒 Suas chaves são armazenadas de forma criptografada. Elas nunca aparecem em texto claro após salvas — apenas os últimos 4 caracteres são exibidos como confirmação. A criptografia usa Fernet (AES-128-CBC + HMAC), processada no servidor Python. Use o botão "Testar" para verificar se a chave está funcional antes de rodar análises.
